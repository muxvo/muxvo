Quick Start

一分钟内启动 Muxvo。

1. 下载安装

bash # macOS (Apple Silicon / Intel) curl -fsSL https://muxvo.app/install | sh

2. 启动

bash muxvo

3. 添加 CLI 工具

Muxvo 自动检测已安装的 AI CLI。当前支持：

  • Claude Code (Anthropic)
  • Codex (OpenAI)
  • Gemini CLI (Google)

Features

Tiling Terminal Grid

平铺终端网格

全屏平铺管理多个终端 session。拖拽调整大小，点击切换焦点，每个面板独立运行。

  • 支持 1-9 个终端面板
  • 拖拽边框调整大小
  • 每个面板独立追踪工作目录
Chat History Search

聊天历史搜索

跨 session 全文搜索所有 AI 对话记录。按日期、项目、工具筛选，一键导出。

  • 全文搜索所有历史对话
  • 按日期、项目、工具筛选
  • 导出为 Markdown 或 JSON
Skill Marketplace + AI Scoring

Skill 市场 + AI 评分

发现和安装社区 Skill 包。每个 Skill 都经过 AI 质量自动评估，帮你快速判断质量。

  • 浏览和搜索社区 Skill 包
  • 一键安装和自动更新
  • AI 驱动的质量评分系统
