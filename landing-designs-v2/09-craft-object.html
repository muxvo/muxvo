Features

核心功能

01

平铺终端

全屏网格布局，多终端并行。2x2、3x1、自定义，每个面板独立运行。

02

聊天历史

跨 Session 全文搜索 AI 对话，按时间和项目过滤，支持导出。

03

文件浏览

工作台内置文件树面板，不离开终端即可浏览项目结构。

04

配置编辑

可视化管理 CLAUDE.md 和各 CLI 配置文件，告别手动编辑。

05

Skill 市场

社区 Skill 包生态。AI 自动评分，一键安装，版本管理。

06

多 CLI 管理

Claude Code、Codex、Gemini CLI 统一检测与管理。

[ 平铺终端 2x2 ]

全屏平铺终端

像平铺窗口管理器一样操控终端。灵活网格布局，每个面板独立运行不同的 CLI。

  • 灵活网格：2x2, 3x1, 自定义
  • 独立 Resize 和 Focus
  • 前台进程状态显示
[ 聊天历史搜索 ]

搜索每段 AI 对话

全文搜索所有 AI CLI 对话记录，跨 Session 跨工具，秒级返回。

  • 全文搜索，跨工具跨 Session
  • 时间线与项目多维过滤
  • 一键导出
