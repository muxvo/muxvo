### 4.1 功能 A：终端创建与关闭

---

#### TERM_L2_01_create_terminal

| 属性 | 值 |
|------|-----|
| 优先级 | P0 |
| 功能 | A - 全屏平铺终端管理 |
| 前置条件 | 应用已启动，终端数 < 20 |

| # | 操作 | 预期结果 |
