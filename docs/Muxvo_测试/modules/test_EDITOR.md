### 1.2 图片附件生命周期

```
  ┌──────────┐    粘贴/拖拽图片     ┌───────────┐     处理完成     ┌───────────┐
  │  无附件   ├───────────────────►│  附加中    ├───────────────►│  已附加    │
  │          │                     │(保存临时   │                │(显示缩略图)│
  └──────────┘                     │ 文件中)    │                └─────┬─────┘
       ▲                           └───────────┘                      │
       │                                                         Enter/
       │                                                         Cmd+Enter
       │            发送完成                                         │
       │  ┌──────────────────────────────────────────────────────────┘
       │  │
       │  ▼
  ┌────┴──────┐
  │  发送中    │

  │ /路径注入) │
```

### 1.3 模式切换触发信号

```
  检测信号                       切换方向              模式
  ─────────────────────────────────────────────────────────
  \x1b[?1049h (Alternate ON)   ──────────►   RawTerminal
  \x1b[?1049l (Alternate OFF)  ◄──────────   RichEditor
  手动快捷键                    ◄─────────►  双向切换
  进程名检测 (fallback)         ──────────►   RawTerminal
                                              (vim/nano/less等)
```

### 1.4 按键行为分类

```
  ┌─────────────────────────────────────────────────────────────┐
  │                    RichEditor 模式按键路由                    │
  ├─────────────────┬───────────────────────────────────────────┤
  │ 按键             │ 行为                                      │
  ├─────────────────┼───────────────────────────────────────────┤
  │ 普通字符输入      │ 编辑器接收                                 │
  │ Shift+Enter     │ 编辑器换行（不发送）                         │
  │ Enter           │ 编辑器发送内容到 PTY                        │
  │ Cmd+Enter       │ 编辑器发送内容到 PTY（同 Enter）            │
  │ Ctrl+C          │ 穿透到终端（编辑器内容保留不清空）             │
  │ Ctrl+Z          │ 穿透到终端（编辑器内容保留不清空）             │
  │ Ctrl+D          │ 穿透到终端（编辑器内容保留不清空）             │
  │ Ctrl+V          │ 粘贴内容到编辑器（文本/图片）                 │
  │ Up Arrow        │ 输入历史召回（RE3）                         │
  │ Tab             │ 斜杠命令补全（RE3）                         │
  └─────────────────┴───────────────────────────────────────────┘
```

### 1.5 图片发送方式路由

```
  检测前台进程
       │
       ├── Claude Code (CC)
       │       │
       │       ├── 首选: 写入剪贴板 → 发送 \x16 (Ctrl+V字符)
       │       │         → 触发CC原生粘贴
       │       │
       │       └── Fallback: 插入文件路径
       │                     /tmp/muxvo-images/{uuid}.png
       │
       ├── Gemini CLI
       │       └── 插入文件路径
       │
       ├── Codex CLI
       │       └── 插入文件路径
       │
       └── 普通 Shell / 其他
               └── 插入文件路径
```

### 1.6 文本→PTY 转发协议

```
  检测前台进程
       │
       ├── Claude Code (CC)
       │       └── 多行文本: 每行用 \x1b\r 连接
       │           单行文本: 直接发送 + \r
       │
       ├── bash / zsh (shell)
       │       └── 直接发送原始文本 + \r
       │
       └── 其他 AI CLI
               └── 按各工具协议适配
```
